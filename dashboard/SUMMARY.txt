==========================================
TROUBLESHOOTING
==========================================

Issue: "Failed to load status"
Solution: Check if backend API is running at localhost:8000

Issue: Port 3000 in use
Solution: Kill process or use PORT=3001 npm run dev

Issue: npm install fails
Solution: Delete node_modules, run npm cache clean --force, retry

See QUICKSTART.md for more troubleshooting.
